# MBE Research Portfolio

Public-facing research portfolio for MBE growth, characterization, modeling, and workflow tools developed by James Rushing for quantum-materials research.

This repository is a curated public export. The private research workspace contains raw growth notes, instrument exports, private databases, local working notebooks, and internal facility material. Those records are intentionally not mirrored here.

## Public Themes

- Epitaxial growth workflows for antimonide quantum materials.
- RHEED-informed growth planning and in-situ decision support.
- Cadtronics/k.p band-structure analysis for transport-oriented heterostructure design.
- Growth database and provenance tooling for turning growth days into structured, queryable records.
- Reproducible plotting, XRD/RSM analysis, and literature-screening workflows.

## Repository Status

This public repository is intentionally selective. Some tools are stable utilities; others are active research workflow components or beta demos. Status labels are included on portfolio and demo pages so reviewers can tell what is mature, what is under active development, and what is included as provenance.

# Public Agent Cards

These agent specifications describe research-assistant workflows used to keep MBE growth, literature screening, plotting, and database work reproducible.

Public-ready agents:

- `lit-search-agent`
- `plotting-analysis-agent`
- `database-agent`

Public summaries:

- `growth-day-assistant-public`
- `characterization-agent-public`

The public summaries intentionally avoid facility-specific procedures, access details, safety-critical steps, and private raw data.

# Characterization Agent Public Summary

Status: `Public summary`

The characterization assistant helps plan, troubleshoot, and interpret AFM, XRD, XRR, RSM, and related materials-characterization workflows. The public version focuses on reusable interpretation logic, not facility training.

## Public Scope

- distinguish measurement geometry issues from materials conclusions
- document scan type, optics, alignment assumptions, and analysis choices
- connect AFM, XRD/XRR/RSM, and microscopy evidence
- flag when conclusions require additional measurements
- preserve raw-data provenance without publishing raw private data

## Private Boundary

Facility-specific instrument procedures, access details, lock/unlock steps, and safety-critical operating instructions are not exported here.

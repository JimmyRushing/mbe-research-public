Public Boundary

The generated HTML is public-facing and self-contained. Raw Cadtronics sweep files remain private; the reusable generator scripts are linked in the public tools folder.

How This Was Generated

The phase diagram was generated from Cadtronics/k.p sweep outputs using phase_diagram_explorer.py. The public repo includes the generator and the generated standalone HTML source. Raw simulation sweep files are intentionally not mirrored publicly. 

python tools/cadtronics/phase_diagram_explorer.py <cadtronics-sweep-dir> --out docs/demos/cadtronics/interactive_pure_GaSb_phase_diagram.html

# Public Research Notebooks

Cleaned Jupyter notebooks for public inspection. These notebooks preserve the workflow logic while excluding raw instrument exports, private lab paths, and scratch notes.

## Notebooks

- [RSM reciprocal-coordinate workflow](rsm_reciprocal_coordinate_workflow.ipynb): public-safe version of the GaSb/GaAs(111)A reciprocal-space-map workflow used for qx/qz plotting, log-intensity visualization, and relaxation-guide overlays.
- [RIOs growth-rate conversion](rios_growth_rate_conversion.ipynb): public-safe version of the RHEED-intensity-oscillation calculator used to convert oscillation-derived growth rates into composition, temperature, and timing estimates.

## Public Boundary

The notebooks use toy or placeholder inputs where needed. Raw RSM spreadsheets, instrument exports, and facility-specific operational details remain private.

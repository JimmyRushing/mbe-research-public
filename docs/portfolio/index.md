# Research Portfolio

This portfolio collects selected public evidence for MBE growth, characterization, modeling, and workflow tooling.

## Main Threads

### Epitaxial Growth And Characterization

Selected presentations document the development of III-V antimonide heterostructures, including GaSb/GaAs(111)A, GaInSb/InAs/AlSb constituent-layer optimization, Sb(111)-oriented growth strategy, and RHEED/AFM/XRD/TEM-informed process learning.

[View selected presentations](presentations.md)

### Selected Research Figures

Sanitized public figure examples show representative XRD/XRR analysis, thickness/fringe interpretation, rocking-curve extraction, and AFM/Nomarski morphology feedback. Raw scans, full sample histories, exact run context, and unredacted process details remain private.

[View selected research figures](selected-research-figures.md)

### Growth Database And Provenance Tools

The growth-database workflow converts MBE control-point exports into queryable records, detected growth intervals, readable views, timeline visualizations, stack schematics, and QC summaries. The public demo uses curated or synthetic examples; real raw exports and private growth notes remain outside the public repo.

[View growth-database demo plan](../demos/growth-database/index.md)

[View AI-compatible synthesis framing](ai_compatible_synthesis_framing.md)

### Sb(111) Growth Strategy

The Sb(111) strategy uses matched comparisons across source species, template state, substrate-temperature window, nominal thickness, and RHEED exposure state to identify conditions for continuous low-roughness films.

[View Sb(111) growth strategy](sb111_growth_strategy.md)

### Cadtronics/k.p Modeling

Cadtronics/k.p tools analyze Hamiltonian-based dispersion outputs, extract hybridization gaps, validate against Schmid/Wurzburg-style phase diagrams, and generate interactive phase-diagram visualizers for growth/design decisions.

[View Cadtronics demo](../demos/cadtronics/)

### RHEED Planning And Growth-Day Decision Support

RHEED tools compute double-mounted sample setpoints and generate an interactive zone-axis finder for (001)/(111) mounted samples. These tools support reproducible in-situ monitoring plans and cleaner growth-day records.

[View RHEED demo plan](../demos/rheed/index.md)

### Reproducible Analysis Workflows

The public tool map includes XRD plotting utilities, public notebooks, literature search tools, and agent specifications for reproducible analysis, database maintenance, and literature triage.

[View tool map](tools.md)

## Status Labels

- `Stable utility`: small reusable script with clear inputs.
- `Active workflow tool`: used in current research; expects domain context.
- `Research prototype`: useful but still being consolidated or documented.
- `Beta`: public enough to show, with known limitations.
- `Public selected slides`: curated public deck or excerpt.
